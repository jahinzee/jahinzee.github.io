# Hi, I'm Jahin…

…but you can call me `jahinzee`[^1]. I'm a software developer and graphic designer based in Sydney, currently also doing a Bachelor's in IT. I spend most of my time studying, but when I'm not, you'll find me:

- [writing code for random things](https://github.com/jahinzee), (usually in Python, although I am also learning Go),
- doing [graphic design](/graphics),
- attending [hackathons](https://github.com/ctrl+alt+elit3/habitasis) (we won state-level prize with this one!),
- taking [photos](/photos) of places I go to, and
- enjoying YouTube video essays, trainspotting, [music](/playlist), and whatever else I might find interesting on any given Thursday.

I'm also a supporter of Free and Open Source and [Local-First](https://www.inkandswitch.com/local-first/) software, although I don't always restrict myself to them.
