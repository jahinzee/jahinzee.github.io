# Hi, I'm Jahin…

…but you can call me `jahinzee`.

I'm a software developer and graphic designer based in Sydney, currently also doing a Bachelor's in IT. I spend most of my time studying, but when I'm not, you'll find me:

- [writing code for random things](https://github.com/jahinzee) – usually in Python or shell script,
- doing [graphic design](/graphics),
- attending [hackathons](https://github.com/ctrl+alt+elit3/habitasis) (we won a state-level prize with this one!),
- taking [photos](/photos) of places I visit, and
- enjoying YouTube video essays, trains and [rail infrastructure](/articles/barangaroo), [music](/playlist), and whatever else I might find interesting on any given Tuesday.

I'm also a supporter of Free and Open Source and [Local-First](https://www.inkandswitch.com/local-first/) software, although I don't always restrict myself to them.

Check out what's in store with the links at the top, and drop into my [Mastodon](https://mastodon.online/@jahinzee) to see what I'm up to. :)
